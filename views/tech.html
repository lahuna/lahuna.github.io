The Data Services

Lahuna web-apps integrate with social media APIs using REST to allow users to upload, post, organize, manipulate, and search their social media data.

Our Web Technology

For our web solutions we use what is called the MEAN stack. M for MongoDB database, E for Express.js, A for Angular, and N for Node.js. These frameworks and services allow developers to code in Javascript for the front-end and back-end. MongoDB even stores data as JSON (JavaScript Object Notation) documents that interface with Javascript beautifully.

The Building Blocks of the Internet

HTML is at the core of any website. We use the latest HTML standard, HTML5 which offers greater multimedia support as well as cross-platform capabilities for different browsers and devices. We also incorporate Responsive Web Design which allows our web apps to remain visually pleasing on varying screen resolutions, from smartphones to TVs. Most of our apps utilize API interfaces via REST to send and receive data from third party providers such as Google. REST is a standardized approach to sharing data across the Web.
