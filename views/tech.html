Google

The Data Services

Lahuna web apps integrate with Google's web services using REST APIs to allow users to upload, organize, manipulate, and search their data stored on Google's cloud storage.

Microsoft

The Development and Hosting Environment

At Lahuna we use Microsoft's Visual Studio 2013 integrated development environment (IDE) for all aspects of our software development process. From writing our HTML5 web pages to coding our backend C# code. From building a Web API REST service to publishing our web site on Azure. Microsoft provides us all the tools we need to get the job done.

Web

The Building Blocks of the Internet

HTML is at the core of any website. We use the latest HTML standard, HTML5 which offers greater multimedia support as well as cross-platform capabilities for different browsers and devices. We also incorporate Responsive Web Design which allows our web apps to remain visually pleasing on varying screen resolutions, from smartphones to TVs. Most of our apps utilize API interfaces to send and receive data from third party providers such as Google. Some of the APIs use application specific web service interfaces. Others use REST which is a more standardized approach to sharing data across the Web.
