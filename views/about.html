Welcome to Lahuna

Lahuna is built with the latest HTML5 web standards and technology. HTML5 bridges the gap between web apps and platform-specific native apps, creating a more standardized approach to writing software.

Cross-Platform Apps

At Lahuna, our core belief is that one version of software should run on any type of web-enabled device without the need to rewrite code for specific operating systems and hardware.

Behind the Scenes

We write our web apps using frameworks like Angular and Bootstrap so we can code faster and more efficiently. Both our front-end and back-end are written in Javascript.

What's Coming Next

We will be releasing more web apps that will run on any device with a browser. Right now we are working on an app called Blitz that will allow you to post to all your favorite social media sites at once.

Who We Are

Lahuna's Mission

We leverage social media APIs (Application Programming Interface) to allow users to manage, organize, and utilize their services and data in a more powerful and meaningful way.

What We Do

  • Use web services to their full potential.

La-Who-Na??

Why do we call ourselves "Lahuna"

  • It is a combination of Laguna and Kahuna.
  • Laguna: Our favorite seaside town Laguna Beach, California.
  • Kahuna: An important figure in Hawaiian culture.
  • It just sounds Hawaiian.
  • And everybody loves Hawaii, right?
